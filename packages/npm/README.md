# solodeveling

[English documentation](https://github.com/ohmiler/solodeveling#readme) |
[เอกสารภาษาไทย](https://github.com/ohmiler/solodeveling/blob/main/README.th.md)

One agent. The right amount of process. Solodeveling is a single-agent-first software
delivery workflow that stays quiet for small changes and adds durable planning,
verification, security, and release controls as risk grows.

Install the complete skill suite into the current project with one command:

    npx solodeveling install

Then start a fresh agent session. Use solodeveling check after installation or an
upgrade, and solodeveling uninstall to remove only unchanged managed files.

The npm package has no install lifecycle script and no runtime npm dependencies. On
explicit invocation it downloads the exact-version native executable from the
immutable Solodeveling GitHub Release, verifies its bundled SHA-256 and size, caches
it, and runs it without a shell. Unsupported platforms fail closed and show the
Python installation fallback.

Read the full overview, comparison, and support boundaries:
